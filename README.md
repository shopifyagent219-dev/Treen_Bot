# 🤖 TrendBot - Telegram Trend Monitoring Bot

A Telegram bot that tracks trending topics from news and social media.

## 🚀 Features

- 📰 Trending News from RSS feeds
- 📊 Top Reddit posts
- 🔍 Custom trend search
- 📈 Popular topics analysis
- 🔄 Real-time updates

## 🛠️ Tech Stack

- Python 3.11
- python-telegram-bot
- Requests
- Railway Deployment
